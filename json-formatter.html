JSON Quick Guide

JSON Rules:
  • Data in name/value pairs
  • Data separated by commas
  • Objects in curly braces {}
  • Arrays in square brackets []
  • Strings in double quotes
  • No trailing commas
Data Types:
  • String: "text"
  • Number: 123, 45.67
  • Boolean: true, false
  • Array: [1, 2, 3]
  • Object: {"key": "value"}
  • null
